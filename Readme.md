This is Java application to consume predict with tsukamoto fuzzy.

## Feature
 Show the chart of your input
 Calculate the prediction

### Build
#### Requirement
- Java 8
- Netbeans

#### Generate Jar
- Run command
  ```
  $ .javac \src\fuzzy\Frame.java
  $ .java \src\fuzzy\Frame.java
  ```
- Your jar will located at `dist`
